> Test project using Cypress to test the Public Api GoRest

### 🏠 [PUBLIC-API Gorest](https://gorest.co.in/)

## Pre-requisites

```sh
Node JS (NPM)
Cypress
```

## Run tests

```sh
'npx cypress run'
or
'npx cypress open' to open Cypress to run the tests in interactive mode.
```
